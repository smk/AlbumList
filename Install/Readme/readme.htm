Introduction

Album List is a Winamp general plugin that takes a directory tree and turns it into a list of albums. This allows easy navigation among albums stored in different directories, and those stored on CDs.

Major functionality includes:

  • multiple profiles
  • multiple search directories
  • different sorting options
  • different display formats
  • skinnable
  • dockable
  • shortcut keys
  • language packs

Requirements

  • Winamp 5, Winamp 2.9
  • Windows XP, Windows 2000 (Windows Me not tested)

    • Using Album List

    After installing Album List, to setup the search paths that you want Album List to look for your albums, go to the "Media" preferences page (1) by right-clicking on the Album List window. To add search paths, click on the "Add" button (2) and browse to the "top level" directory that you have your music files stored. Repeat this process for each different location. You don't have to add every single directory that you have, only the "top level" directories as Album List will search the sub-directories for you by default. Alternately, you can just drag and drop the directories from Windows Explorer onto Album list.

    Next you would want to specify what file types (3) you want Album List to scan. By default, Album List will scan all for most of the popular music file types such as MP3, MPC, MP+, MPP, WMA. The list is seperated by commas.

    And finally, you can customize how Album List does its scanning by changing the options (4) in this properties page.

    After setting up where to look for your albums, it is now time to tell Album List how you want them displayed, go to the second tab labelled "Title" to set that up.

    Directory style "auto-detect" should be good for most of the cases. In case it doesn't work for you, you could try the other predefined directory styles

    To create additional profiles, go to Winamp Preferences, select Album List (1) and press New (2).

    Uninstall

    To uninstall Album List, follow these steps: (the recommended way)

    1. Open Windows Control Panel
    2. Select "Add/Remove Programs"
    3. Click on "Album List for Winamp v2.0 (remove only)"
    4. Select "Remove"
    5. Follow the instructions from the uninstall program

    Questions & Answers

    Is classic skins supported?

    Yes, it does support classic skins. Due to architectural differences, it does not use the playlist editor's skin anymore. It uses the new skin bitmap that's defined in Winamp 2.9, namely Gen.bmp & Genex.bmp. The same ones that Media Library uses.

    How to sort albums?

    Click on the column header or right-click on the "Name" header.

    Is multiple sorting levels supported?

    Yes. For example, if you want to sort by artist, then year, click "sort by year" first then "sort by artist".

    How to skip a directory?

    Put a ~ in front of the directory.

    Is Free Format MP3 supported?

    Yes, but it is in its preliminary stage. Put Free Format MP3=1 in [Main] of Gen_m3a.ini to enable it.

    Can I rename a profile?

    Yes, just not through the UI yet. Put Name=Rock in [Profile 0] of Gen_m3a.ini to change it.
